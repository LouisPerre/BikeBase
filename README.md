## Installation

We try to keep our library update all the time, so try it out:

### NPM
```sh
    npm i bikebase
```
### Yarn
```sh
    yarn add bikebase
```

## Import

To import and use our library in your project you can use this :
```javascript
import * from 'bikebase'
```
